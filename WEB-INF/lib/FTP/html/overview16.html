Overview
   

The Sftp class provides an easy to use client application programming interface for transferring files between a client and SSH (Secure SHell) server using the sftp subsystem. Classes may be found in the com.jscape.inet.sftp package.

This section is divided into the following topics.

Connecting to an SSH server
Capturing SFTP related events
Setting your directory
Retrieving a directory listing
Uploading files
Downloading files
Remote file operations
