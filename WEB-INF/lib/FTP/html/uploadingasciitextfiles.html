Uploading ASCII text files
   

To upload ASCII text files you must set the transfer mode to ASCII using the Ftp#setAscii method. All files with relative paths will be uploaded from your current local directory.

Example

// turn off auto transfer mode detection if enabled
ftp.setAuto(false);

// set transfer mode to ASCII
ftp.setAscii();

// upload all files in current local directory
ftp.mupload("*.*");
