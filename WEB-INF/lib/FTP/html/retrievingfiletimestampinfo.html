Retrieving file timestamp information
   

You can retrieve the date a file was last modified using the Ftp#getFileTimestamp method. All filenames are relative to your current remote directory.

Example

// get last modified date of image.gif
Date date = ftp.getFileTimestamp("image.gif"); 

Note

The Ftp#getFileTimestamp method utilizes a command that is supported by a majority of FTP servers however is not officially recognized by RFC 959 so may not be supported on some FTP servers.
